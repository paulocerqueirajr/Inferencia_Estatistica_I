Exercício 1 Seja \(X_1, \ldots, X_n\) uma amostra aleatória da v.a. \(X \sim \text{Poisson}(\theta)\). Considere os seguintes estimadores para \(\theta\):

\[ \hat{\theta}_1 = \bar{X} \quad \text{e} \quad \hat{\theta}_2 = \frac{X_1 + X_2}{2} \]

Verifique se os estimadores são não viciados e consistentes, e compare seus EQMs.

Exercício 2 Seja \(X_1, \ldots, X_n\) uma amostra aleatória da v.a. \(X \sim \text{Bernoulli}(\theta)\). Considere os seguintes estimadores para \(\theta\):

\[ \hat{\theta}_1 = 2\bar{X} \quad \text{e} \quad \hat{\theta}_2 = \frac{1}{2} \]

  1. Verifique se os estimadores são não viciados e consistentes.

  2. Faça um gráfico dos seus EQMs e compare-os.

Exercício 3 Seja \(X_1, \ldots, X_n\) uma amostra aleatória da v.a. \(X \sim \text{Normal}(\mu, 4)\). Considere os seguintes estimadores para \(\theta\):

\[ \hat{\theta}_1 = \bar{X} \quad \text{e} \quad \hat{\theta}_2 = 8 \] a. Verifique se os estimadores são não viciados e consistentes.

  1. Faça um gráfico dos seus EQMs e compare-os.

Exercício 4 Seja \(X\) uma única observação da \(\text{Bernoulli}(\theta)\). Considere os estimadores:

\[ T_1(X) = X \quad \text{e} \quad T_2(X) = \frac{1}{2} \] a. \(T_1(X)\) e \(T_2(X)\) são estimadores não viciados para \(\theta\)? \

  1. Calcule o erro quadrático médio de \(T_1(X)\) e \(T_2(X)\).

Exercício 5 Seja \(X_1, \ldots, X_n\) uma a.a. da v.a. com f.d.p. \(f(x|\theta) = \exp{-(x - \theta)}, \, x > \theta, \, \theta > 0\). Considere:

\[ \hat{\theta}_1 = \bar{X} \quad \text{e} \quad \hat{\theta}_2 = X_{(1)} \]

  1. Verifique se os estimadores são não viciados.

  2. Faça um gráfico dos EQMs e compare-os.

Exercício 6 Seja \(X_1, \ldots, X_n\) uma a.a. da v.a. com f.d.p. \(f(x|\theta) = \theta x^{\theta - 1}\), \(0 < x < 1\), \(\theta > 0\). Mostre que:

\[ \hat{\theta} = \bar{X} \] é um estimador não-viciado para \(\theta\).

Exercício 7 Seja \(X_1, \ldots, X_n\) uma a.a. da v.a. \(X \sim \text{Normal}(\mu, 1)\). Mostre que \(T\) é não viciado para \(g(\mu) = \mu^2\).

Exercício 8 Seja \(X_1, \ldots, X_n\) uma a.a. com f.d.p. \(f(x|\theta) = \frac{2x}{\theta^2}\), \(0 < x < \theta\), \(\theta > 0\). Considere:

\[ \hat{\theta}_1 = c_1 \bar{X} \quad \text{e} \quad \hat{\theta}_2 = c_2 X_{(n)}. \]

  1. Verifique se são não-viciados.

  2. Compare os EQMs.

Exercício 9 Seja \(X_1, \ldots, X_n \sim \text{Normal}(0, \sigma^2)\) e \(V^2 = \sum_{i=1}^{n} X_i^2\), considere:

\[ \hat{\sigma}^2_c = c V^2 \] a. Encontre o EQM.

  1. Determine \(c\) que minimiza o EQM.

Exercício 10 Sejam \(X_1, \ldots, X_n\) uma a.a. da v.a. \(X\), com função de densidade:

\[ f(x) = \theta x^{\theta - 1} I_{(0,1)}(x), \quad \theta > 0 \]

  1. Mostre que \(X\) pertence à Família Exponencial.
  2. Encontre o limite inferior da variância dos estimadores não viciados de \(\theta\).
  3. Encontre uma estatística suficiente para \(\theta\).
  4. Calcule o valor esperado desta estatística.

Exercício 11 Seja \(X_1, \ldots, X_n\) uma amostra aleatória da distribuição:

\[ f(x) = \exp \left\{ -(x - \theta) \right\} I_{(\theta, \infty)}(x), \quad \theta > 0 \]

  1. Obtenha uma estatística suficiente para \(\theta\).
  2. Obtenha um estimador não viciado que seja função da estatística suficiente.

Exercício 12 Mostre que as seguintes distribuições pertencem à família exponencial:

  1. \(\text{Gama}(\alpha, \gamma)\) com \(\alpha\) e \(\gamma\) desconhecidos.
  2. \(\text{Gama}(\alpha, \gamma)\) com \(\alpha\) conhecido e \(\gamma\) desconhecido.
  3. \(\text{Beta}(a, b)\) com \(a\) e \(b\) desconhecidos.
  4. \(\text{Beta}(a, b)\) com \(a\) conhecido e \(b\) desconhecido.
  5. \(\text{Poisson}(\theta)\).
  6. Binomial Negativa com número de sucessos \(r\) conhecido e \(0 < p < 1\) desconhecido.

Exercício 13 Para cada item da questão 14, encontre uma estatística suficiente para os parâmetros de interesse.

Exercício 14 Seja \(X_1, \ldots, X_n\) uma amostra da distribuição:

\[ f(x_i | \theta) = \frac{1}{2i\theta}, \quad -i(\theta - 1) < x_i < i(\theta + 1), \ \theta > 0 \]

Encontre uma estatística suficiente bidimensional para \(\theta\).

Exercício 15 Seja \(X_1, \ldots, X_n \sim \text{Gama}(2, 1/\theta)\).

  1. Mostre que \(X\) pertence à Família Exponencial.
  2. Obtenha uma estatística suficiente para \(\theta\).

Exercício 16 Seja \(X_1, \ldots, X_n \sim N(0, \sigma^2)\).

  1. Encontre o limite inferior da variância dos estimadores não viciados de \(\sigma^2\).
  2. Obtenha uma estatística suficiente para \(\sigma^2\).
  3. Obtenha um estimador não viciado que seja função da estatística suficiente.
  4. Verifique se este estimador é eficiente.

Exercício 17 Seja \(X_1, \ldots, X_n \sim \text{Bin}(2, \theta)\).

  1. Encontre o limite inferior da variância dos estimadores não viciados de \(\theta\).

  3. Obtenha um estimador não viciado que seja função da estatística suficiente.

Exercício 18 Sejam \(Y_1, \ldots, Y_n\) independentes com \(Y_i \sim N(\beta x_i, \sigma^2)\).

  1. Obtenha uma estatística suficiente para \(\beta\) e \(\sigma^2\).
  2. Obtenha o ENVVUM para \(\beta\) e \(\sigma^2\).

Exercício 19 Seja \(X_1, \ldots, X_n\) da v.a. com densidade:

\[ f(x | \theta) = \frac{2x}{\theta^2}, \quad 0 < x < \theta, \ \theta > 0 \]

Obtenha uma estatística suficiente para \(\theta\).

Exercício 20 Seja \(X_1, \ldots, X_n \sim \text{Bernoulli}(\theta)\).

  1. Obtenha o ENVVUM para \(\theta\).
  2. Obtenha o ENVVUM para \(\theta(1 - \theta).\).

Exercício 21 Seja \(X_1, \ldots, X_n\) com densidade:

\[ f(x | \mu, \sigma) = \frac{1}{\sigma} \exp\left\{-\frac{(x - \mu)}{\sigma}\right\}, \quad \mu < x < \infty, \ 0 < \sigma < \infty \]

Encontre uma estatística bidimensional para o vetor \((\mu, \sigma)\).
