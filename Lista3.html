Exercício 1 Em estudos de genética, o modelo Binomial é frequentemente utilizado, exceto quando a observação \(x = 0\), pois é impossível de ocorrer. Nestes casos, a amostragem será realizada a partir da seguinte distribuição truncada:

\[P(X = x)= \begin{cases} \binom{m}{x} p^x (1 - p)^{m-x} / \left[ 1 - (1 - p)^m \right], & \text{se } x \in \{1, 2, \dots, m\} \\ 0, & \text{caso contrário}. \end{cases}\]

Encontre o EMV para \(p\) para quando \(m = 2\) e o tamanho da amostra é \(n\).

Exercício 2 Sejam \(X_1, \dots, X_n\) uma a.a. da v.a. \(X \sim \text{Exp}(\lambda)\).

Encontre o EMV para \(\lambda\).

Exercício 3 Seja \(X_1, \dots, X_n\) uma a.a. de \(X \sim \text{Unif}(\theta, \theta + 1)\).

  1. Obtenha um estimador via método de momentos.

  2. Obtenha um estimador via método da máxima verossimilhança.

Exercício 4 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X \sim \text{Gama}(2, 1/\theta)\).

  1. Obtenha o EMV para \(\theta\).

  2. Obtenha a distribuição para grandes amostras do estimador obtido no item a).

Exercício 5 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X\), com função de densidade dada por:

\[ f(x \mid \theta) = \frac{x + 1}{\theta (\theta + 1)} \exp\left\{-\frac{x}{\theta}\right\}, \quad x > 0, \theta > 0. \]

  1. Obtenha o EMV para \(\theta\).

  2. Obtenha a distribuição para grandes amostras do estimador obtido no item a).

  3. Obtenha um estimador via método de momentos.

Exercício 6 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X \sim \text{Poisson}(\theta)\). Seja \(g(\theta) = \exp\{-\theta\}\).

  1. Obtenha o EMV para \(g(\theta)\).

  2. Obtenha a distribuição para grandes amostras do estimador obtido no item a).

Exercício 7 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X \sim N(0, \sigma^2)\).

  1. Obtenha o EMV para \(\sigma^2\).

  2. Obtenha o EMV para \(\sigma\).

Exercício 8 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X\), com função de densidade dada por:

\[ f(x \mid \theta) = \theta (1 + x)^{-(1+\theta)} I_{(0, \infty)}(x), \quad \theta > 0. \]

  1. Obtenha o EMV para \(1/\theta\).

  2. Obtenha a distribuição para grandes amostras do estimador obtido no item a).

  3. Obtenha um estimador via método de momentos.

Exercício 9 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X \sim \text{Poisson}(\theta)\). Seja \(g(\theta) = (1 + \theta) \exp\{-\theta\}\).

Obtenha o EMV para \(g(\theta)\).

Exercício 10 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X\), com função de densidade dada por:

\[ f(x \mid \theta) = \theta x^{\theta - 1} I_{(0, 1)}(x), \quad \theta > 0. \]

  1. Encontre o EMV para \(\theta\).

  2. Encontre o EMV para \(g(\theta) = \frac{\theta}{1 + \theta}\).

Exercício 11 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X\), com função de densidade dada por:

\[ f(x \mid \theta) = \frac{2x}{\theta^2}, \quad 0 < x < \theta, \theta > 0. \]

  1. Obtenha o EMV para \(g(\theta) = \theta + \theta^2\).

  2. Obtenha um estimador para \(\theta\) via métodos de momentos.

Exercício 12 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X\), com função de densidade dada por:

\[ f(x \mid \theta) = \frac{\theta}{x^2}, \quad x > \theta, \theta > 0. \]

  1. Obtenha o EMV para \(\theta\).

  2. Obtenha o EMV para \(g(\theta) = E\left(\frac{1}{X}\right)\).

Exercício 13 Seja \(X_1, \dots, X_n\) uma amostra aleatória obtida a partir da distribuição:

\[f(x) = \exp\{-(x - \theta)\} I_{(\theta, \infty)}(x), \quad \theta > 0.\]

  1. Obtenha o EMV para \(\theta\).

  2. Obtenha um estimador para \(\theta\) via métodos de momentos.

Exercício 14 Seja \(Y_1, \dots, Y_n\) variáveis aleatórias independentes com \(Y_i \sim N(\alpha + \beta x_i, \sigma^2)\), em que \(x_i\) é conhecido para todo \(i = 1, \dots, n\).

Encontre o EMV para \(\alpha\), \(\beta\) e \(\sigma^2\).

Exercício 15 Seja \(X_1, \dots, X_n\) uma amostra aleatória da v.a. \(X \sim \text{Ber}(\theta)\).

  1. Encontre o EMV para \(\text{Var}(X)\).

  2. Obtenha a distribuição para grandes amostras do estimador obtido no item a).
